Electric Field Simulator

How to use

Left click to create a positive charge. Right click to create a negative charge (If on mobile use the options)

Use the options at the bottom to change the physics timescale, image resolution, # of physics subsamples, friction, charge, trails, and static charges.

(Note, this will run on firefox, but is optimized for chrome and is a lot more performant, especially with resolution)
